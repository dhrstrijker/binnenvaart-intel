# Navisio Marketing Pack

This folder contains ready-to-use marketing assets for Navisio:

- `logo-circle.svg`: Circular logo avatar for social/profile use
- `logo-square.svg`: Square logo for app stores, docs, and embeds
- `favicon.svg`: Browser favicon source
- `website-banner.svg`: Hero/banner image for website sections
- `email-banner.svg`: Wide banner for newsletters and transactional emails
- `polar-post-purchase-copy.md`: Product marketing blocks for Polar checkout success/upsell pages
- `dutch-only/`: Volledig Nederlandstalige conceptset (banners + Polar copy)

## Notes

- All assets are SVG for easy resizing without quality loss.
- Brand palette used:
  - Cyan: `#06B6D4`
  - Slate dark: `#0F172A`
  - Slate mid: `#1E293B`
  - White: `#FFFFFF`
- Recommended exports (if needed):
  - Logo circle/square: 1024x1024 PNG
  - Favicon: 32x32 and 16x16 PNG + `.ico`
  - Website banner: 1600x600 PNG
  - Email banner: 1200x400 PNG
